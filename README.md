# 🧮 Modern Calculator – Codesoft Internship Task 2

## 📌 Project Description
This is a **modern GUI calculator** built using Python's Tkinter library as part of my **Codesoft Internship**.

- Subtraction ➖
- Multiplication ✖️
- Division ➗
